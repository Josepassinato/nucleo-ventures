## 🚀 O que é Núcleo Ventures?

Núcleo Ventures é um **framework open-source de IA autônoma** que permite criar uma diretoria de agentes inteligentes para rodar qualquer negócio (loja, serviço, produto físico/digital) de forma 100% autônoma.

Seus agentes:
- ✅ Decidem e executam ações
- ✅ Pagam fornecedores
- ✅ Contratam e demitem
- ✅ Anunciam e otimizam
- ✅ Parecem 100% humanos para o mundo externo

---

## ✨ Principais Features

### 🤖 Diretoria de Agentes com "Alma"

- **9 Agentes Especializados**: CEO, CMO, CFO, COO, CPO, CHRO, Coach, Analista, Otimizador
- **Perfis Psicológicos Realistas**: Cada agente tem temperamento, estresse, proatividade e medo de falhar
- **Supervisão Cruzada**: Um vigia o outro; erro grave bloqueia APIs
- **Sistema de Leaderboard**: Ranking 0-10 baseado em KPIs, proatividade e cultura

### 🎯 Dashboard Executivo

- **2 Versões de Design**: V1 Premium (verde-esmeralda) e V2 Ousado (azul-neon)
- **Organograma Vivo**: Visualize status de cada agente em tempo real
- **Timeline de Ações**: Histórico automático de tudo que os agentes fizeram
- **Kill Switch**: Botão de emergência para parar tudo imediatamente
- **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile

### 🛠️ Ferramentas Avançadas

- **Navegação Autônoma**: Agentes navegam sites reais, preenchem formulários, fazem screenshots
- **Orquestração CrewAI**: Framework profissional para coordenar agentes
- **Banco de Dados PostgreSQL**: Persistência completa de dados
- **Autenticação OAuth**: Login seguro com Manus
- **API REST completa**: Integre com qualquer sistema

### 📦 Onboarding Simples

- **Wizard de 6 Passos**: Configure tudo em 5 minutos
- **Links Diretos para APIs**: Saiba exatamente onde obter credenciais
- **Validação em Tempo Real**: Teste conexão com cada serviço
- **Script de 1 Linha**: Instale tudo com um comando

### 🔒 Segurança e Conformidade

- ✅ **LGPD** (Brasil) - Conformidade total com Lei Geral de Proteção de Dados
- ✅ **CCPA** (EUA) - Conformidade com California Consumer Privacy Act
- ✅ **Criptografia AES-256**: Credenciais protegidas
- ✅ **Auditoria Completa**: Tudo é logado e criptografado
- ✅ **Backup Automático**: A cada 12 horas
- ✅ **Kill Switch**: Parar tudo em caso de emergência

## 🚀 Como Começar

### Opção 1: Instalação Rápida (Recomendado)

```bash
curl -fsSL https://raw.githubusercontent.com/Josepassinato/nucleo-ventures/main/install.sh | bash
```

Tempo total: **5 minutos**

### Opção 2: Docker

```bash
docker run -p 3000:3000 josepassinato/nucleo-ventures:latest
```

### Opção 3: Desenvolvimento Local

```bash
git clone https://github.com/Josepassinato/nucleo-ventures.git
cd nucleo-ventures
pnpm install
pnpm dev
```
