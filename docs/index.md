# 🚀 Núcleo Ventures - Autonomous AI Directory Framework

**Crie uma diretoria de IA que roda seu negócio 100% autônoma.**

Seus agentes decidem, executam, pagam, contratam, anunciam e otimizam — tudo sozinho, com cara humana pro mundo externo.

---

## ✨ O Que Você Consegue Fazer

### 🤖 Diretoria Autônoma
- 9 agentes especializados (CEO, CMO, CFO, COO, CPO, CHRO, Coach, Analista, Otimizador)
- Cada um com temperamento, estresse, proatividade e medo de falhar
- Supervisão cruzada: um vigia o outro
- Sistema de leaderboard com recompensas

### 🎯 Integração Total
- **Brasil**: Mercado Livre, Stripe, Twilio, WhatsApp, Gmail
- **EUA**: Amazon, Stripe, Twilio, WhatsApp, Gmail
- **Global**: Google Gemini (grátis!), DocuSign, 2Captcha

### 📊 Dashboard Executivo
- 2 versões de design (Premium + Ousado)
- Organograma vivo com status em tempo real
- Timeline de ações automáticas
- Kill Switch de emergência

### 🔒 Segurança Total
- LGPD (Brasil) + CCPA (EUA)
- Criptografia AES-256
- Auditoria completa
- Backup automático

## 🚀 Começar em 5 Minutos

### Instalação Rápida

```bash
curl -fsSL https://raw.githubusercontent.com/Josepassinato/nucleo-ventures/main/install.sh | bash
```

### Ou com Docker

```bash
docker run -p 3000:3000 josepassinato/nucleo-ventures:latest
```

### Ou Localmente

```bash
git clone https://github.com/Josepassinato/nucleo-ventures.git
cd nucleo-ventures
pnpm install && pnpm dev
```
